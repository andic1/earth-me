是一个基于 React 和 Three.js 构建的未来派全息地球探索界面。它结合了沉浸式的 3D 可视化技术与AI 的强大生成能力，为用户提供了一个探索地球地理、生态和人文知识的智能窗口。
“本项目由阿里云ESA提供加速、计算和保护”以及附右图➡️<img width="7534" height="844" alt="image" src="https://github.com/user-attachments/assets/e51349d6-cea6-4e8e-9175-9bdddd17a9b3" />

本项目面向 Web 端的地理可视化与交互场景，提供了一个可通过摄像头手势直接操控的三维地球界面，用户无需学习复杂操作即可完成地球旋转、缩放和区域选择，具有较强的实用性与可扩展性，可用于课堂教学、地理科普展示以及互动展览等多种应用场景。在创意性方面，作品将手势识别、3D 地球渲染与大模型地理知识分析深度融合，实现了“挥手转动地球、点选位置即可获得 AI 解读”的沉浸式交互体验，相比传统鼠标点击式地图更具未来感和参与感。技术层面上，项目采用 React + Vite + TypeScript 构建现代前端工程体系，使用 Three.js 完成地球纹理贴图、光照与交互拾取，借助 MediaPipe 实现实时手势识别，并通过 OpenAI 兼容协议对接 Qwen 大模型服务，整体架构清晰、模块划分合理，既体现了前端工程化能力，也展示了对 3D 图形、实时交互和大模型 API 集成等多项技术的综合掌握。
